# To Do List
<p>Uma to-do list básica em formato de prompt para ajudar a gerenciar suas tarefas e organizar seu trabalho. Com ela, você pode adicionar novas atividades, marcar as concluídas e excluir tarefas finalizadas, facilitando o controle das suas responsabilidades diárias. </p>

## Funcionalidades
<ul>
    <li>Adicionar Tarefas: Inclua novas tarefas com título e descrição. </li>
    <li>Visualização de Tarefas: Veja uma lista de todas as suas tarefas.</li>
    <li>Excluir Tarefas: Remova tarefas que não são mais necessárias. </li>
</ul>

## Como Usar
Clone o Repositório: git clone https://github.com/GuttenbergJr/to_do_app.git <br>
Execute o Projeto: no console do execute o comando "python main.py"

Sinta-se à vontade para abrir issues e pull requests para melhorias e correções!
